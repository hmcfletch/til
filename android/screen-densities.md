# Screen Densities

Different screen sizes require different size assets. This table gives some basic guidelines to the different sizes you need to generate.

|             | dpi | ratio |
| ----------- | --- | -----:|
| **ldpi**    | 120 | 0.75  |
| **mdpi**    | 160 | 1     |
| **hdpi**    | 240 | 1.5   |
| **xhdpi**   | 320 | 2     |
| **xxhdpi**  | 480 | 3     |
| **xxxhdpi** | 640 | 4     |
